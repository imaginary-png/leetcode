# Climbing Stairs
## Easy
### https://leetcode.com/problems/climbing-stairs/
#### O(n)

Use memoization  

Record the possible combinations at step m, where n > m > 0   

Recursively counting back from n, with (n-1) (n-2)  

Like a tree
